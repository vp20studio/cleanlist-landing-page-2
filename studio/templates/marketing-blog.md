# Marketing Blog Template

Use this template for educational content targeting marketing teams and demand gen professionals.

## Frontmatter

```yaml
---
title: "[How to/Why/What] + [Marketing Outcome]"
description: "[150-160 chars: Educational hook + key takeaway]"
date: "YYYY-MM-DD"
category: "marketing"
tags: ["marketing", "demand-gen", "relevant-topic"]
author:
  name: "Cleanlist Team"
  role: "Content Team"
featuredImage:
  src: "/images/blog/marketing-topic.png"
  alt: "Descriptive alt text"
keywords: ["primary keyword", "secondary keyword", "LSI keyword"]
---
```

## Structure

### 1. The Setup (100-150 words)
- Start with a common scenario or question
- Establish relevance quickly
- Preview what they'll learn

Example:
> "You've built the perfect nurture sequence. The copy's sharp, the timing's dialed in. But your deliverability is tanking. Sound familiar? Let's talk about the data quality problem hiding in your marketing stack."

### 2. The Context (200-250 words)
- Why this topic matters now
- Industry trends or shifts
- What's changed recently
- Set up the educational content

### 3. Core Educational Content (500-700 words)
- Main insights organized into 3-5 sections
- Use H3 headings for each major point
- Include examples and data
- Mix theory with practical application

#### Section Structure:
```
### [Insight/Concept Title]

[Opening statement about this concept]

[2-3 paragraphs explaining the concept]

**In practice**: [Real-world example or application]
```

### 4. Common Mistakes (150-200 words)
- 3-5 pitfalls to avoid
- Brief explanation of each
- What to do instead

### 5. Action Items (100-150 words)
- Specific next steps
- Prioritized by impact
- Link to relevant resources

### 6. Key Takeaways (75-100 words)
- 3-4 bullet point summary
- Reinforces main message
- Easy to screenshot/share

### 7. Soft CTA (50 words)
- Natural connection to Cleanlist
- Value-first approach
- Not pushy

---

## Word Count Target: 1,500-2,000 words

## Internal Links to Include:
- Link to relevant product page
- Link to use case page (/use-cases/agencies or similar)
- Link to related educational content
- Link to free trial

## SEO Checklist:
- [ ] Educational keyword in H1 ("how to", "guide", "why")
- [ ] Primary keyword in first paragraph
- [ ] Secondary keywords distributed in H2/H3s
- [ ] FAQ schema opportunity at end
- [ ] Long-tail keyword variations

## Tone Reminders:
- Educational but not condescending
- Back claims with data
- Acknowledge marketing complexity
- Focus on outcomes, not just tactics
